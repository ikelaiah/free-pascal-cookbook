# External Systems

Your Pascal program does not have to live on an island. This section shows you
how to exchange data, call web services, store information, run system commands,
and serve content of your own.

## Pick a Connection

- **Exchange structured data:** Read [JSON](parsing-json.md) and
  [XML](working-with-xml.md).
- **Talk to the web:** Make [HTTP requests](http-requests-advanced.md), add
  [API authentication](api-authentication.md), or launch a small
  [web server](webserver.md).
- **Keep useful information:** Work with [SQLite databases](sqlite-databases.md)
  and [environment-based configuration](env-vars-config.md).
- **Reach the operating system and beyond:** Run [system calls](system-calls.md)
  or [send an email](sending-emails.md).

Choose the connection your project needs and build outward from there. The
recipes are independent, so there is no required reading order.

!!! tip "Need the tools first?"

    [Set up the Free Pascal Compiler and Lazarus IDE](../basics/install.md), then
    come back and connect your first program to the outside world.
