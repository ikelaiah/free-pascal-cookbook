# Core Tasks

Now we're cooking. This section turns the language basics into useful programs:
organise code, work with collections, shape strings and numbers, handle dates,
read and write files, catch errors, and bundle files into archives.

## What Do You Want to Make Easier?

- **Build a well-shaped program:** Start with [code structure](code-structure.md),
  then split the work into [functions and procedures](funcs-procs.md).
- **Put data to work:** Explore [arrays](arrays.md), [lists](lists.md), and
  [strings](strings.md).
- **Deal with everyday input:** Try [command-line parameters](cmd-line-params.md),
  [dates and times](dates.md), or [regular expressions](regex.md).
- **Work with the filesystem:** Learn about [paths and filenames](dir-path-filename.md),
  [file handling](file-handling.md), and [zip archives](zip-and-unzip.md).

Each page is a standalone recipe, so follow your curiosity or jump straight to
the problem you need to solve.

    come back and choose a recipe.
